该beamer模板为仿照李文威老师的著作《代数学方法：卷二》风格制作，代码部分许多参考自李文威老师的[源代码](https://github.com/wenweili/AlJabr-2)，有一些是照抄，有一些由于书籍模板和beamer模板存在差异和不兼容做了对应的修改。


模板大量使用衬线字体（思源宋体、仿宋、Source Serif 4等），色调以黑白灰为主，彩色使用较为克制，仅有紫罗兰色和两种较淡的黄白色作为区分或提示用。定理环境根据用途不同使用了不同字体，具体可见cls文件。


右侧为导航条，模拟网页的滑块；移除了默认模板右下角的跳转按钮，改为在左上角增加两个浅灰色按钮，上（下）按钮表示跳转到当前节（section）的第一页（最后一页）。
